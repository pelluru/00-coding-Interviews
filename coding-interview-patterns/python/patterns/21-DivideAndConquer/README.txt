
Divide & Conquer — Core (3 Problems)
====================================
Includes:
1) count_inversions — merge-sort based inversion count (O(n log n))
2) max_subarray_divide — max subarray via (sum, pref, suf, best) combine
3) majority_element — Boyer–Moore voting (O(n), O(1))

Usage:
  python divide_conquer_core_all_examples.py

Or run individually:
  python divide_conquer_example1_count_inversions.py
  python divide_conquer_example2_max_subarray.py
  python divide_conquer_example3_majority_element.py
