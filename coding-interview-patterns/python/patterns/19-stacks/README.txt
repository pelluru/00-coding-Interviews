
Stack & Monotonic Stack Bundle
==============================
Includes five classic stack problems:
1) Next Greater Element — monotonic decreasing stack
2) Daily Temperatures — wait days for warmer temperature
3) Largest Rectangle in Histogram — monotonic increasing stack
4) Stock Span — span of days with price <= today's
5) Valid Parentheses — bracket matching

Usage:
  python stack_monotonic_all_examples.py
  python stack_monotonic_all_examples.py 1 3 5

Or run individually:
  python stack_monotonic_example1_next_greater.py
  python stack_monotonic_example2_daily_temperatures.py
  python stack_monotonic_example3_largest_rectangle.py
  python stack_monotonic_example4_stock_span.py
  python stack_monotonic_example5_valid_parentheses.py
