
Math Counting Examples Bundle
=============================
Includes five core counting utilities/patterns:
1) modpow (binary exponentiation)
2) nCr_mod (combinations modulo prime)
3) gcd / lcm
4) nPr_mod (permutations modulo prime)
5) Applications: m^k, subset count (2^n-1), lcm over list

Usage:
  python math_counting_all_examples.py
  python math_counting_all_examples.py 1 3 5

Or run individually:
  python math_counting_example1_modpow.py
  python math_counting_example2_nCr_mod.py
  python math_counting_example3_gcd_lcm.py
  python math_counting_example4_nPr_mod.py
  python math_counting_example5_applications.py
