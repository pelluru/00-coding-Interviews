
Backtracking Examples Bundle
============================
Includes five classic backtracking problems:
1) Subsets (Power Set)
2) Permutations (distinct)
3) Combination Sum (allow reuse)
4) N-Queens
5) Letter Combinations of a Phone Number

Usage:
  python backtracking_all_examples.py
  python backtracking_all_examples.py 1 3 5

Or run individually:
  python backtracking_example1_subsets.py
  python backtracking_example2_permutations.py
  python backtracking_example3_combination_sum.py
  python backtracking_example4_nqueens.py
  python backtracking_example5_letter_combinations.py
