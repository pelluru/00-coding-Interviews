
DFS Examples Bundle

Files:
- dfs_all_examples.py                    # Run all or selected examples
- dfs_example1_graph_traversal.py       # Example 1 (graph DFS traversal)
- dfs_example2_binary_tree_traversal.py # Example 2 (tree preorder DFS)
- dfs_example3_connected_components.py  # Example 3 (connected components)
- dfs_example4_islands_grid.py          # Example 4 (island counting on grid)
- dfs_example5_all_paths_backtracking.py# Example 5 (all paths with backtracking)

Usage:
  python dfs_all_examples.py
  python dfs_all_examples.py 1 3 5

  # Or run examples individually:
  python dfs_example1_graph_traversal.py
  python dfs_example2_binary_tree_traversal.py
  python dfs_example3_connected_components.py
  python dfs_example4_islands_grid.py
  python dfs_example5_all_paths_backtracking.py
