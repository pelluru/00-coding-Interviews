
Hashing Examples Bundle
=======================
Includes five classic hashing problems:
1) Two Sum (hash map of seen values)
2) Group Anagrams (hash by canonical key)
3) First Unique Character (hash counting)
4) Longest Consecutive Sequence (hash set)
5) Subarray Sum Equals K (prefix sum hashing)

Usage:
  python hashing_all_examples.py
  python hashing_all_examples.py 1 3 5
