
Matrix Algorithms Bundle
========================
Includes five matrix problems with clean, in-place or O(1) extra patterns:
1) Search in sorted matrix — O(m+n) staircase
2) Rotate image 90° — transpose + reverse rows, O(1) extra
3) Flood fill — DFS/BFS region recolor
4) Spiral order — layer traversal with four bounds
5) Set matrix zeroes — O(1) extra space using first row/col markers

Usage:
  python matrix_algorithms_all_examples.py
  python matrix_algorithms_all_examples.py 1 3 5

Or run individually:
  python matrix_algorithms_example1_search_sorted.py
  python matrix_algorithms_example2_rotate_image.py
  python matrix_algorithms_example3_flood_fill.py
  python matrix_algorithms_example4_spiral_order.py
  python matrix_algorithms_example5_set_zeroes.py
